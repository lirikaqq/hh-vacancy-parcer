# Парсер вакансий с уведомлениями в Telegram

![Python](https://img.shields.io/badge/python-3.11-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

Скрипт на Python, который каждые N минут (или по расписанию) проверяет новые вакансии на hh.ru по заданным критериям и отправляет уведомления в Telegram. Полезен для активного поиска работы в IT-сфере.

## Особенности

- **Автоматический парсинг** – собирает вакансии с hh.ru по ключевым словам (например, "AI-автоматизация").
- **Уведомления в Telegram** – новые вакансии приходят прямо в личный чат.
- **Без дубликатов** – скрипт запоминает ID уже отправленных вакансий в JSON-файле.
- **Гибкая настройка** – легко изменить город, поисковый запрос, интервал проверки.
- **Обработка ошибок** – устойчив к временным сбоям сети и API.

## Технологии

- Python 3.11
- requests – работа с API hh.ru и Telegram
- python-dotenv – безопасное хранение токенов
- JSON – хранение состояния

## Установка и запуск

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/lirikaqq/hh-vacancy-parser.git
cd hh-vacancy-parser

2. Установите зависимости
bash
python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt

3. Создайте Telegram-бота и получите токен
В Telegram найдите @BotFather, создайте нового бота.
Скопируйте токен.

4. Узнайте ваш chat_id
Напишите боту любое сообщение.
Перейдите по ссылке: https://api.telegram.org/bot<ТОКЕН>/getUpdates
Найдите поле chat → id.

5. Настройте переменные окружения
Создайте файл .env в корне проекта:

BOT_TOKEN=ваш_токен_бота
CHAT_ID=ваш_chat_id

6. Настройте параметры поиска (опционально)
В файле main.py измените переменные:
SEARCH_TEXT – поисковый запрос (поддерживает OR, кавычки)
AREA – код региона (76 – Ростов-на-Дону, 1 – Москва, 2 – СПб)
FILE_NAME – имя файла для сохранения ID отправленных вакансий

7. Запустите вручную

bash
python main.py

8. Настройте автоматический запуск
Для постоянной проверки (например, каждый час) используйте планировщик задач Windows или cron на Linux.
